# 甜點世界電商網站 [DessertWorld](https://esanking.github.io/DessertWorld/)

### 功能
前台：
* 產品展示
* 產品收藏
* 加入購物車
* 搜尋產品及分類
* 下單結帳

後台：
* 產品管理
* 訂單列表管理
* 優惠碼管理
##
### 使用技術
* Vue.js
* Vue CLI 4
* Vue Router 模擬路由
* Vuex 統一管理狀態
* RWD 響應式畫面
* jQuery 互動效果
* ES6
* Webpack
* ESlint(aribnb) 程式碼管理
* Bootstrap 4 及 SCSS 的應用
* 第三方 Plugin 的運用
  * axios
  * vue-axios
  * vue-router
  * jQuery
  * popper.js
  * Bootstrap 4
  * vue-loading-overlay
  * vee-validate
  * aos
